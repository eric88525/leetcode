## 中位數
+ 將陣列由小到大排序，中間切一半， 左邊的MAX + 右邊的MIN 平均就是答案

## 概念

1. 將A和B陣列，切割成左右兩個群組

```
      left      right
A:   |0-------|i---------|M
      left      right
B:   |0-------|j---------|N
```


2. 為了符合由小大的排序
    + A[i] > b[j-1] and B[j]>M[i-1]
    + 左邊的MAX < 右邊的MIN 
3. 為了符合 左邊群組數量 = 右邊群組數量 :
    + i + j = M-i + N-j
  
  
因此要想辦法找出最合適的 i 和 j
